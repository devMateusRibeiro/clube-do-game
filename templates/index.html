📘 Sobre o Projeto

O Clube de Game é um site que conecta você à RAWG API, uma das maiores bases de dados de jogos do mundo. A ideia é simples: você escolhe seus filtros (ou deixa tudo em aberto) e recebe uma recomendação aleatória com imagens, nota, plataformas e mais.

⚙️ Tecnologias Utilizadas

  • Python + Flask (backend)
  • Tailwind CSS (estilo responsivo)
  • RAWG API (dados dos jogos)
  • JavaScript moderno (UX dinâmica)
